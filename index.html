Browsing Images

This section will help you navigate and browse images within the asset collection.

1. Using Keywords

Keywords, also known as tags, are one of the most powerful tools for navigating the asset collection. Keywords are descriptive words or phrases attached to images to help categorize and find them, regardless of what album they are in.

  • How Keywords Work: An image can have multiple keywords (e.g., 'campus', 'engineering', 'building', 'exterior'), making it discoverable through many different search terms.
  • Finding Images by Keyword: When you view a photo's metadata, you will see a list of its associated keywords. Clicking on any of these keywords will take you to a page that shows all other images with the same keyword. This is a great way to find visually similar images or images related to the same topic.
  • Best Practices for Keywords: To ensure reusability and searchability, we will be using a consistent set of keywords. When uploading or editing images, please choose from the existing keyword list as much as possible to avoid creating duplicates.

2. Searching for Images

The Mines Online Asset Collection provides powerful search capabilities to help you quickly locate specific images.

  • Quick Search Field: Simply type a word or phrase and press Enter. Piwigo will intelligently search the photo's name (title), file name, description, associated tags, and photo ID to find relevant results.
  • Multi-Criteria Search: For more refined and precise searches, you can access the dedicated search page. Here, you can apply various filters to narrow down your results:
    • Search by Word: Similar to the quick search, but with additional options to search for *all* words or *any* of the words you enter.
    • Filter by Tag: Search for photos associated with one or more specific tags.
    • Filter by Album: Limit your search to photos within one or more specific albums, with an important option to include sub-albums in your search.
    • Filter by Author: Search for photos uploaded by a particular author.
  • Searching within a Set: When you are viewing an album or a tag page, you might see a "Search in this set" button or icon. Clicking this will apply the search filters specifically to the photos within that particular album or tag, helping you focus your search.

3. Browsing by Albums

The most common way to browse images in the Mines Online Asset Collection is through Albums.

  • Album Structure: Albums are organized like a directory tree. They can contain sub-albums, allowing for a hierarchical organization of our assets (e.g., "Campus Buildings" > "Green Center" > "Exterior Shots").
  • Accessing Albums: From the Piwigo homepage, you'll see a list of available albums. Click on an album to enter its page and view its contents.
  • Navigation within Albums: You can use the breadcrumb trail at the top to navigate back to parent albums, and pagination links at the bottom to move between pages of photos.

4. Managing Keywords on Existing Assets (Single Edit)

To perform cleanup, add detail, or refine the search terms for a single image, you can edit its metadata directly from the Photo Page.

Workflow for Editing Keywords (Single Image):

  1. Locate the Image: Navigate to the specific image you want to edit (e.g., via search or album browsing).
  2. Access the 'Edit' Option: On the Photo Page, look for and click the 'Edit' icon (a pencil symbol) in the toolbar. This opens the image's metadata editing screen.
  3. Locate the Tag/Keyword Field: Find the dedicated input field for Tags or Keywords.
  4. Manage Tags:
    • Adding Tags: Type new keywords, separating them with commas or pressing Enter.
    • Removing Tags: Click the 'X' next to any existing tag you want to remove.
  5. Consistency Check: Always prioritize using existing, consistent keywords over creating new ones to maintain collection integrity.
  6. Save Changes: Click the 'Save' button at the bottom of the editing screen to apply your changes.

Note: If you need to edit keywords for many images at once, use the Batch Manager instead, which is accessible from the album view toolbar.

Downloading Images

This section will guide you through the process of downloading and organizing images from the Mines Online Asset Collection for use in your projects.

1. How to Download an Image

You can download an image directly from its Photo Page.

  • Navigate to the specific image you wish to download.
  • On the Photo Page, look for the Download Photo icon (usually a down arrow).
  • Clicking this icon will initiate the download of the image to your computer. By default, Piwigo provides the original, highest-quality version of the image, however you can select from other sizes, as well.

2. Tracking Image Usage

To help us efficiently track the use of our image assets across Mines Online, ProEd, and other projects, we will now be using a dedicated album for each course or project.

Your Responsibility: Once an image is used for a course or project, you must associate it with the correct project album.

How to Associate an Image with a Project Album:

  1. Locate the Image: After you have downloaded an image for a project, return to the image's Photo Page in Piwigo.
  2. Access the 'Edit' Option: Look for the 'Edit' option for the photo (pencil-shaped icon).
  3. Find the 'Linked Albums' section: Within the editing window, find the section to associate the image with an album.
  4. Click ‘Add’ to open the album dialog.
  5. Select or Create the Project Album: Choose the correct album for your project. If an album for your project does not exist, you must create one first by clicking the 'Create a new album' link in the dialog, selecting a parent album, and entering a name.
    • Example for a ProEd or Mines Online course: 'EBGN595 FA25' (Course Code + Semester/Year).
    • Example for a HIVE project: 'HIVE: [Project Title]' (e.g., 'HIVE: Oredigger Welcome Video').
    • Example for an internal document: 'Internal: [Document Title]' (e.g., 'Internal: Guide to Piwigo').
  6. Save Changes: Be sure to save your changes to update the image's associations.

Uploading Images

This section will walk you through the process of adding new images to the Mines Online Asset Collection.

1. What to Upload

To maintain a consistent and legally compliant collection, please only upload images that fall into one of the following categories:

  • Content created by staff or students at the Colorado School of Mines.
  • Images that are in the Public Domain.
  • Content licensed under a Creative Commons (CC) license CC0 or CC BY (requiring attribution). Other CC license types are not permitted.
  • Content that has been licensed to us by a third party for internal or external use, unless the license specifically restricts its upload to a centralized image database.

2. Accessing the Upload Feature

To begin uploading images, you'll need to access the administration area of Piwigo. From the left-hand menu, navigate to Photos > Add. This will take you to the Photo Upload page.

3. Choosing or Creating a Destination Album

Before you upload, you must select the album where your photos will be stored. Click the icon next to "Choose an album" on the Photo Upload page. A window will appear where you can select an existing album or create a new one.

Creating a New Project Album:

If you are uploading assets for a new course or project, you must first create a dedicated album for tracking usage. This ensures all project-related assets are organized together.

  1. In the album selection dialog, click the link to 'Create a new album' (or similar button/link).
  2. Select the Album you wish to place the project album inside (e.g., a top-level 'Courses' or 'Projects' album).
  3. Enter the Album name, strictly following our naming conventions:
    • Example for a ProEd or Mines Online course: 'EBGN595 FA25' (Course Code + Semester/Year).
    • Example for a HIVE project: 'HIVE: [Project Title]' (e.g., 'HIVE: Oredigger Welcome Video').
    • Example for an internal document: 'Internal: [Document Title]' (e.g., 'Internal: Guide to Piwigo').
  4. Click 'Create and select.'

If the album already exists, simply click the '+' icon next to it to select it.

4. Selecting and Uploading Your Files

Once your destination album is chosen, you can select the image files from your computer using one of two methods: Click "Add Photos" to open a file browser or Drag and Drop files into the designated gray area on the page. After adding your files, click "Start Upload".

5. Post-Upload Actions

After your images have successfully uploaded, you'll have a few options. We have implemented a specific workflow for adding key metadata that will ensure our images are easy to find and reusable for years to come.

Add Metadata:

  • File Name: Do not edit the file name. The system handles this automatically.
  • Title: Add a concise title for the image. If there is a pre-existing title, use that. Otherwise, a short, descriptive title will suffice.
